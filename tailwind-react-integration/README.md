# Tailwind React Integration

This project demonstrates integrating **Tailwind CSS** with a **React (Vite)** application. It includes reusable components, responsive layouts, hover effects, and transitions.

## Features

- Tailwind CSS setup with Vite
- Responsive Navbar
- Reusable Card components
- Hover effects & transitions
- Mobile-friendly layout
